This is an archive of preset banks made by Stephen Wey/Kymeia/aMUSEd. It includes many banks for current plugins as well as some for those that are probably defunct or hard to find these days such as Augur and Dash EVE 2, but I've included them in case anyone still uses those. Some like the bank for Glitchmachines Polygon only work in version 1. I have not added banks and presets I have made for factory banks of plugins - which includes several for u-he, XILS-Labs and Waldorf, and also Reaktor ensembles I have contributed or co-authored in the Reaktor User Library.
